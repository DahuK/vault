# ICon

`ICons` are glyphs used to indicate important information.

## Properties
| Property | Required | Default value | Type | Description | Example |
|---|---|---|---|---|
| glyph | [x] | `null` | string | The glyph type. | `'cancel-square-outline'` |

## Usage

```javascript
<ICon @glyph="cancel-square-outline" />
```
